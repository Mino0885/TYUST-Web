# TYUST-Web
## 太原科技大学(Taiyuan University of Science and Technology) 管理后台 web版
## Author : Mino0885
## E-mail : Mino0885@outlook.com

- 语言: Java PHP
- 版本控制 : git maven
- 数据库: Mysql
- 基于SSM框架 (Spring-SpringMVC-Mybatis)
- 安全框架暂定为 Spring Security

### 模块划分
> tyust-patent 父项目
> > tyust-controller 控制层
> 
> > tyust-service 业务层
> 
> > tyust-dao 持久层
> 
> > tyust-pojo 存放持久化对象以及vo
>
> > tyust-Utils 存放通用工具类
